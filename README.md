# VOL-API-CALLS
This project contains the methods that call the VOL Backend API

## Prerequisite 
The following technologies should be installed on your system.
* Java JDK 8
* Maven 3

## Technologies
* Java
* Maven


## Deploying

### Local

``mvn install``

### Nexus

``mvn deploy``
